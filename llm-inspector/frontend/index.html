新建检测任务

利用统计学指纹与测试矩阵验证模型真实身份。系统将依据预检测置信度自动分配算力资源。

仅支持 http/https，不允许内网地址
密钥在服务端使用 AES-256-GCM 加密存储，72小时后自动清除
检测流程说明
① 预检测（0-200 token）
20 层渐进探针：HTTP头 → 自我报告 → 身份矩阵 → 知识截止 → 行为偏见 → Tokenizer → 主动提取 → 语义指纹 → 高级提取 → 差分测试 → 工具探针 → 多轮溢出 → 对抗分析 → 多语言攻击 → ASCII艺术 → 间接注入 → 身份碰撞 → 时序侧信道 → Token分布侧信道 → 系统提示词抽取。约60%情况在此阶段直接识别。
② 能力测试
指令遵循、System权重、参数生效、风格采集。置信度达标时提前终止。
③ 分析报告
特征提取 → 4维评分 → 相似度排名（含置信区间）→ 风险等级评估。
