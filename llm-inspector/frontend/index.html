新建检测任务

输入目标 API 信息，系统将自动进行预检测识别，必要时执行完整能力测试。

仅支持 http/https，不允许内网地址
密钥在服务端使用 AES-256-GCM 加密存储，72小时后自动清除
检测流程说明
① 预检测（0-200 token）
HTTP头分析 → 自我报告 → 身份探针 → 知识截止 → 格式偏见。约60%情况在此阶段直接识别。
② 能力测试（~$0.01-0.05）
指令遵循、System权重、参数生效、风格采集。置信度达标时提前终止。
③ 分析报告
特征提取 → 4维评分 → 相似度排名（含置信区间）→ 风险等级评估。
